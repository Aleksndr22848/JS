О проекте ZAMETOCHKI

Это итоговый проект курса JavaScript (DOM) + Node.js, демонстрирующий создание веб-приложения для управления задачами и получения баллов чтобы я мог наконецто отдохнуть.

Frontend:
  • HTML5
  • CSS3 + Bootstrap 5
  • JavaScript ES6+
Backend:
  • Node.js
  • Express.js
  • CORS
  • REST API
Функциональность
Основные возможности:
  • Создание новых задач
  • Редактирование задач
  • Удаление задач
  • Отметка выполненных
  • Добавление заметок
Дополнительно:
  • Статистика задач
  • Адаптивный дизайн
  • Быстрые действия
  • Автообновление
  • Сохранение в БД

  • Многостраничность (3+ страницы)
  • Адаптивная вёрстка Bootstrap
  • Навигация между разделами
  • Динамический список задач
  • Минимум 5 операций с DOM
  • Работа с формами (2+ формы)

  • Express сервер
  • REST API endpoints
  • CRUD операции (GET/POST/PUT/DELETE)
  • Валидация данных
  • Обработка ошибок

  • JSON обмен данными
  • Асинхронная обработка
  • Обратная связь пользователю
  • Обработка сетевых ошибок
